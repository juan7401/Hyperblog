Para ver que tags hay 
    * git tags 

Agregar un tag 

    * git tag -a "nombre del tag" -m "descripcion del tag"

Mostrar referencia del tag

    * git show-ref --tags

enviar el tag al repositorio

    * git push origin --tag

eliminiar tag 

    * git tag -d "nombre del tag"

eliminar referencia 

    * git push origin :refs/tags/"nombre del tag"
